### 预期目标

我们建议使用OS级别上2~3个可能的"攻击向量"（Attack Vector）。要求学生

1. 说明这些向量是如何绕过OS及其他简易的保护机制并造成损害的
2. 设计、部署保护机制以在执行前检出
3. 拓展第2点，设计、部署可以检出跨函数、跨文件的相似攻击模式
4. 如果无法给出自行设计、部署保护机制的明确回答，那么还可以通过部署哪些解决方案来修补所提议的静态解决方案的缺陷？
